1. The area tag attributes

The HTML <area> element defines a hot-spot region on an image, and optionally associates it with a hypertext link. This element is used only within a <map> element. The <area> tag attributes include;

  • shape
  • coords
  • href
  • target
  • alt
  • name
  • rel
  • ping
  • nohref
